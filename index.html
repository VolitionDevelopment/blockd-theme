About Block'd

Block'd is an elegant landing page fit for any Minecraft server website. If you're tired of using Enjin's awful themes or paying high prices for Zenforo themes, you should be interested in Block'd.

Why I made it

Truthfully, I think Minecraft servers should be able to live up to higher standards of web design. Practically every server nowadays is either a crappy-looking Enjin website or a decent-looking (though very expensive) Zenforo website. I aim to fill a gap in the market: a slightly cheaper website template that is elegant, simple, and user-friendly.

Though I would never refuse money to customize a theme for a server
